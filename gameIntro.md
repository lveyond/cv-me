# 游戏中心开发需求文档

## 概述

本文档用于记录游戏中心（Game Center）的功能开发需求、游戏设计思路和实现计划。

## 当前游戏列表

### 1. 交易模拟器 (Trading Simulator)
- **类型**: 新手指导类
- **状态**: ✅ 已实现基础版本
- **功能**: 模拟真实交易场景，学习买卖时机
- **待优化**:
  - [ ] 添加更多交易策略提示
  - [ ] 增加难度等级
  - [ ] 添加交易历史记录

### 2. 策略拼图 (Strategy Puzzle)
- **类型**: 益智类
- **状态**: ✅ 已实现基础版本
- **功能**: 拼出正确的策略代码，锻炼逻辑思维
- **待优化**:
  - [ ] 增加拼图难度（4x4, 5x5）
  - [ ] 添加策略代码预览
  - [ ] 计时挑战模式

### 3. 价格预测 (Price Prediction)
- **类型**: 竞赛类
- **状态**: ✅ 已实现并优化
- **功能**: 观察K线图预测价格走势，挑战预测准确率
- **游戏机制**:
  - 使用K线图（Candlestick Chart）显示价格走势
  - 绿色K线表示上涨（收盘价>开盘价），红色K线表示下跌（收盘价<开盘价）
  - 10轮预测挑战，根据准确率计算分数
  - 图表自动横向滚动，确保最新K线可见
- **已优化**:
  - ✅ 改为K线图显示（箱型指标）
  - ✅ 轮数从5次增加到10次
  - ✅ 图表自动滚动，防止溢出
  - ✅ 添加游戏说明和图例
- **待优化**:
  - [ ] 添加技术指标提示（MA、MACD等）
  - [ ] 多时间周期预测
  - [ ] 排行榜系统
  - [ ] 添加成交量显示

### 4. 挖矿小游戏 (Mining Game)
- **类型**: 点击类/上瘾类
- **状态**: ✅ 已实现并优化
- **功能**: 点击挖矿区域收集代币，升级装备，达成目标通关
- **游戏机制**:
  - 8x8网格（64个方块）挖矿区域
  - 点击任意方块进行挖矿
  - 有15%概率挖不到任何东西（增加挑战性）
  - 购买升级装备提升挖矿力量
  - 目标：收集1000个代币即可通关
- **已优化**:
  - ✅ 挖矿区域扩大（400px高度，8x8=64个方块）
  - ✅ 规则说明可折叠
  - ✅ 添加挖不到的概率机制（15%失败率）
  - ✅ 通关机制（1000代币目标）
  - ✅ 点击反馈优化（成功/失败不同视觉效果）
  - ✅ 修复点击区域问题
- **待优化**:
  - [ ] 添加自动挖矿功能
  - [ ] 更多升级选项和装备
  - [ ] 成就系统
  - [ ] 特殊方块（稀有资源、炸弹等）
  - [ ] 挖矿动画效果增强

### 5. 交易对匹配 (Pair Matching)
